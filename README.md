## 🚀 快速开始

### 环境要求
- Node.js >= 18
- Rust >= 1.70
- 操作系统：macOS, Windows, Linux

### 安装依赖
```bash
npm install
```

### 开发运行
```bash
npm run tauri:dev
```

### 构建应用
```bash
npm run tauri:build
```

## 📱 使用方法

### 基本操作
1. **启动应用** - 运行后会显示专业的深色主题界面
2. **首次启动** - 跟随向导配置权限和基本设置
3. **选择转录模型** - 在"转录模型"页面选择合适的AI模型
4. **配置音频设备** - 在"常规设置"中选择麦克风设备
5. **开始录音** - 点击录音按钮或使用快捷键
6. **查看结果** - 转录结果会实时显示并保存到历史记录

### 高级功能
- **AI提示管理** - 配置智能处理链，支持转录优化、邮件写作等
- **文件转录** - 拖拽音视频文件到"转录文件"页面
- **历史管理** - 在"历史记录"页面查看和管理所有转录
- **高级快捷键** - 使用增强版快捷键编辑器配置复杂快捷键
- **权限中心** - 统一管理所有系统权限
