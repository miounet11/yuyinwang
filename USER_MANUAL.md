## 快速开始

### 首次使用

1. **启动应用**
   打开 Recording King，首次启动会进入引导页面。

2. **配置权限**（macOS 必需）
   前往"权限管理"页面，授予以下权限：
   - **麦克风权限**（必需）：用于录音
   - **辅助功能权限**（必需）：用于文本注入
   - **输入监控权限**（可选）：用于全局快捷键

3. **选择转录模型**
   前往"听写模型"页面，选择适合您的模型：
   - **在线模型**：需要配置 API Key，准确度高
   - **本地模型**：需要下载，完全离线，保护隐私

4. **设置快捷键**
   前往"快捷键设置"，选择您喜欢的快捷键组合（推荐：右 ⌘ 或 右 ⌥）

5. **开始使用**
   按下快捷键说话，松开后自动转录并注入到当前应用！

## 功能介绍

Recording King 提供 8 个主要功能页面：

### 1. 语音输入

**核心功能**：实时录音和语音转文字

- 点击麦克风按钮开始录音
- 实时显示录音时长和波形动画
- 停止后自动转录为文字
- 支持一键复制或注入到当前应用
- 显示当前使用的转录模型

**使用场景**：
- 快速记录会议纪要
- 语音输入长文本
- 口述文章或邮件

### 2. 常规设置

**界面配置**：
- **录音显示样式**：面板模式 / 刘海模式
- **外观主题**：跟随系统 / 深色 / 浅色
- **界面语言**：跟随系统 / 简体中文 / English

**行为配置**：
- 登录时启动
- 在程序坞中显示
- 在状态栏中显示
- Escape 取消录音

**麦克风优先级**：
- 拖拽调整多个麦克风的使用顺序
- 设备断开时自动切换到下一个可用设备

### 3. 快捷键设置

**快捷键选择**：
- 预设快捷键：右 ⌘、右 ⌥、右 ⇧、右 ⌃、Fn 等
- 自定义快捷键：录制任意组合键

**激活模式**：
- **按住或切换**：自动检测按住时长或快速点击
- **切换**：点击开始，再次点击停止
- **按住**：按下录音，松开停止（推荐）
- **双击**：快速按两次开始录音

**测试区域**：
- 提供文本框测试快捷键是否正常工作

### 4. 听写模型

**在线模型**（需要网络和 API Key）：
- **LuYinWang Transcribe**（免费，推荐）：准确度 ★★★★★，速度 ★★★★
- **GPT-4o mini**（OpenAI）：准确度 ★★★★★，速度 ★★★★，最准确
- **Nova-3**（Deepgram）：准确度 ★★★★，速度 ★★★★★，最快（开发中）

**本地模型**（完全离线，保护隐私）：
- **Whisper Tiny**（75 MB）：准确度 ★★，速度 ★★★★★
- **Whisper Base**（148 MB）：准确度 ★★★，速度 ★★★★
- **Whisper Small**（488 MB）：准确度 ★★★★，速度 ★★★，本地推荐
- **Whisper Medium**（1.5 GB）：准确度 ★★★★，速度 ★★
- **Whisper Large v3**（3.1 GB）：准确度 ★★★★★，速度 ★
- **Large v3 Turbo**（1.6 GB）：准确度 ★★★★★，速度 ★★★，本地最佳

**模型操作**：
- 在线模型：点击"配置 Key"输入 API Key
- 本地模型：点击"下载"按钮，下载完成后点击"使用"

### 5. 转录文件

**支持格式**：
- 音频：MP3、WAV、M4A、FLAC、OGG
- 视频：MP4、MOV、M4V、WebM

**操作流程**：
1. 点击选择文件或拖拽文件到上传区域
2. 点击"开始转录"按钮
3. 等待转录完成（显示进度和用时）
4. 查看结果，支持复制或注入

**使用场景**：
- 转录录音文件
- 提取视频中的语音内容
- 批量处理音频文件

### 6. AI 提示

**功能说明**：配置 AI 提示词和自动化动作

**支持的动作类型**：
- 🔍 Google 搜索
- 🚀 启动应用
- ❌ 关闭应用
- 💬 ChatGPT 对话
- 🤖 Claude 对话
- 📺 YouTube 搜索
- 🌐 打开网站
- ⚡ Apple 快捷指令
- ⌨️ Shell 命令
- ⌨️ 模拟按键

**使用示例**：
- 创建"搜索"提示：语音说"搜索天气"，自动打开 Google 搜索
- 创建"打开应用"提示：语音说"打开微信"，自动启动微信
- 创建"AI 助手"提示：语音说"帮我写邮件"，自动调用 ChatGPT

### 7. 历史记录

**功能特性**：
- 查看所有转录历史
- 搜索历史内容
- 显示转录时间、时长、模型、准确度
- 区分录音转录和文件转录
- 支持复制、注入、删除操作
- 一键清空所有历史

**时间显示**：
- 刚刚、X 分钟前、X 小时前
- 昨天 HH:MM
- MM月DD日 HH:MM

### 8. 权限管理

**系统权限配置**：

| 权限 | 类型 | 用途 |
|------|------|------|
| 🎙 麦克风权限 | 必需 | 录制音频和语音识别 |
| ♿ 辅助功能权限 | 必需 | 系统集成和文本注入 |
| ⌨️ 输入监控权限 | 可选 | 全局快捷键功能 |

**授权步骤**：
1. 点击"授权"按钮
2. 系统会打开"系统设置"
3. 找到对应的隐私设置项
4. 勾选 Recording King
5. 返回应用，权限状态会自动更新

## 操作指南

### 场景 1：快速语音输入

**目标**：在任意应用中使用语音输入文字

**步骤**：
1. 在任意应用中，将光标定位到输入框
2. 按住快捷键（如右 ⌘）
3. 对着麦克风说话
4. 松开快捷键
5. 等待 1-3 秒，文字自动出现在输入框中

**提示**：
- 说话清晰，避免环境噪音
- 使用"按住"激活模式体验最佳
- 如果转录不准确，可以切换到更高准确度的模型

### 场景 2：转录会议录音

**目标**：将会议录音文件转换为文字

**步骤**：
1. 前往"转录文件"页面
2. 点击上传区域或拖拽音频文件
3. 确认当前模型（建议使用 GPT-4o mini 或 Whisper Large）
4. 点击"开始转录"
5. 等待转录完成
6. 复制结果或注入到文档中

**提示**：
- 长音频文件转录时间较长，请耐心等待
- 本地模型完全离线，适合处理敏感内容
- 转录结果会自动保存到历史记录

### 场景 3：配置本地 Whisper 模型

**目标**：使用完全离线的本地模型

**步骤**：
1. 前往"听写模型"页面
2. 点击"本地"筛选按钮
3. 选择合适的模型（推荐 Whisper Small 或 Large v3 Turbo）
4. 点击"下载"按钮
5. 等待下载完成（显示进度条）
6. 点击"使用"按钮切换到该模型

**提示**：
- 首次下载需要时间，请保持网络连接
- 下载后完全离线使用，无需 API Key
- 模型文件较大，确保有足够存储空间

### 场景 4：自定义快捷键

**目标**：设置个性化的快捷键组合

**步骤**：
1. 前往"快捷键设置"页面
2. 在快捷键下拉菜单中选择"录制快捷键..."
3. 在弹出的窗口中，按下您想要的组合键
4. 确认组合键有效（至少一个修饰键 + 一个普通键）
5. 点击"保存"按钮

**提示**：
- 避免使用系统快捷键（如 ⌘+C、⌘+V）
- 推荐使用右侧修饰键，不影响左手操作
- 在测试区域验证快捷键是否正常工作

### 场景 5：配置 API Key

**目标**：使用在线模型（LuYinWang 或 OpenAI）

**步骤**：

**配置 LuYinWang Token**：
1. 访问 [record-to-text.com](https://record-to-text.com) 注册账号
2. 获取您的 JWT Token
3. 在 Recording King 中前往"听写模型"页面
4. 点击"录音王 Token"徽章或选择 LuYinWang 模型
5. 输入 Token（格式：eyJ0eXAi...）
6. 点击"保存并使用"

**配置 OpenAI API Key**：
1. 访问 [platform.openai.com](https://platform.openai.com) 获取 API Key
2. 在 Recording King 中前往"听写模型"页面
3. 点击"OpenAI Key"徽章或选择 GPT-4o mini 模型
4. 输入 API Key（格式：sk-...）
5. 点击"保存并使用"

**提示**：
- Token 和 Key 会安全存储在本地
- LuYinWang 提供免费额度
- OpenAI 按使用量计费

## 常见问题 FAQ

### 基础问题

**Q1: Recording King 是免费的吗？**
A: 应用本身免费，但部分在线模型需要 API Key（按使用量付费）。本地 Whisper 模型完全免费且离线使用。

**Q2: 支持哪些语言？**
A: 支持 99 种语言，包括中文、英文、日文、韩文等。Whisper 模型支持最多语言。

**Q3: 转录准确度如何？**
A: 取决于模型选择：
- GPT-4o mini：准确度最高（★★★★★）
- Whisper Large v3：本地模型中最准确（★★★★★）
- 清晰发音和安静环境可提高准确度

**Q4: 数据安全吗？**
A:
- 本地模型：完全离线，数据不上传
- 在线模型：通过加密连接发送到服务商（OpenAI、LuYinWang）
- API Key 安全存储在本地

### 功能问题

**Q5: 快捷键不工作怎么办？**
A:
1. 检查"权限管理"中的"输入监控权限"是否已授权
2. 确认快捷键没有与其他应用冲突
3. 尝试更换快捷键组合
4. 重启应用

**Q6: 文本注入失败怎么办？**
A:
1. 检查"辅助功能权限"是否已授权
2. 确认光标在可输入的文本框中
3. 某些应用（如终端）可能不支持文本注入
4. 可以使用"复制"功能手动粘贴

**Q7: 如何提高转录速度？**
A:
- 使用速度更快的模型（如 Whisper Tiny、Nova-3）
- 本地模型比在线模型更快（无网络延迟）
- 确保电脑性能良好，关闭不必要的后台程序

**Q8: 可以同时使用多个模型吗？**
A: 不可以，每次只能使用一个模型。但可以随时在"听写模型"页面切换。

**Q9: 历史记录会占用多少空间？**
A: 历史记录仅存储文字，占用空间很小。可以定期在"历史记录"页面清空。

**Q10: 支持实时字幕吗？**
A: 当前版本不支持实时字幕，但可以使用"按住说话"模式实现准实时转录。

### 技术问题

**Q11: 本地模型下载失败怎么办？**
A:
1. 检查网络连接
2. 确保有足够的存储空间
3. 尝试重新下载
4. 检查防火墙设置

**Q12: 麦克风无法识别怎么办？**
A:
1. 检查"麦克风权限"是否已授权
2. 在"常规设置"中调整麦克风优先级
3. 在系统设置中确认麦克风正常工作
4. 尝试重新插拔外接麦克风

**Q13: 应用崩溃或卡顿怎么办？**
A:
1. 重启应用
2. 检查系统资源占用
3. 清空历史记录
4. 重新安装应用

**Q14: 如何卸载应用？**
A:
1. 退出 Recording King
2. 将应用拖到废纸篓
3. 删除配置文件（可选）：`~/Library/Application Support/com.recordingking.app`

## 故障排除

### 问题 1：快捷键无响应

**症状**：按下快捷键没有任何反应

**可能原因**：
- 输入监控权限未授权
- 快捷键与其他应用冲突
- 应用未在后台运行

**解决方案**：
1. 前往"权限管理"页面，确认"输入监控权限"已授权
2. 在"快捷键设置"页面更换快捷键
3. 确认应用在状态栏或程序坞中显示
4. 重启应用

### 问题 2：转录结果不准确

**症状**：转录的文字与实际说话内容差异较大

**可能原因**：
- 环境噪音过大
- 发音不清晰
- 模型选择不当
- 麦克风质量差

**解决方案**：
1. 在安静环境中录音
2. 说话清晰，语速适中
3. 切换到更高准确度的模型（如 GPT-4o mini 或 Whisper Large）
4. 使用质量更好的麦克风
5. 调整麦克风距离（10-30cm 为佳）

### 问题 3：文本注入失败

**症状**：转录完成后文字没有自动输入到应用中

**可能原因**：
- 辅助功能权限未授权
- 目标应用不支持文本注入
- 光标未在输入框中

**解决方案**：
1. 前往"权限管理"页面，确认"辅助功能权限"已授权
2. 确认光标在可输入的文本框中
3. 对于不支持注入的应用，使用"复制"功能手动粘贴
4. 在"常规设置"中调整注入延迟时间

### 问题 4：本地模型下载缓慢

**症状**：下载本地 Whisper 模型速度很慢或中断

**可能原因**：
- 网络速度慢
- 服务器繁忙
- 防火墙拦截

**解决方案**：
1. 使用稳定的网络连接
2. 避开高峰时段下载
3. 检查防火墙和代理设置
4. 如果下载中断，重新点击"下载"按钮继续

### 问题 5：应用启动失败

**症状**：双击应用图标后无法启动

**可能原因**：
- 系统版本过低
- 应用文件损坏
- 权限问题

**解决方案**：
1. 确认 macOS 版本 ≥ 10.13
2. 重新下载并安装应用
3. 在"系统设置 → 隐私与安全性"中允许应用运行
4. 右键点击应用，选择"打开"（首次运行）

### 问题 6：API Key 无效

**症状**：输入 API Key 后提示无效或无法使用

**可能原因**：
- Key 输入错误
- Key 已过期
- 账户余额不足
- 网络连接问题

**解决方案**：
1. 仔细检查 Key 是否完整复制（包括前缀 sk- 或 eyJ）
2. 登录服务商网站确认 Key 状态
3. 检查账户余额
4. 确认网络连接正常
5. 重新生成 Key

### 问题 7：录音无声音

**症状**：录音过程中没有声音波形，转录结果为空

**可能原因**：
- 麦克风权限未授权
- 麦克风被其他应用占用
- 麦克风硬件故障
- 系统音量设置问题

**解决方案**：
1. 前往"权限管理"页面，确认"麦克风权限"已授权
2. 关闭其他使用麦克风的应用（如 Zoom、Skype）
3. 在系统设置中测试麦克风
4. 检查麦克风音量设置
5. 尝试使用其他麦克风

### 问题 8：历史记录丢失

**症状**：之前的转录记录突然消失

**可能原因**：
- 数据库文件损坏
- 应用重新安装
- 手动删除了配置文件

**解决方案**：
1. 检查是否误操作"清空"按钮
2. 历史记录存储在本地数据库，重装应用会清空
3. 定期导出重要的转录结果
4. 暂无自动备份功能，建议手动保存重要内容
